\# Secure Azure Infrastructure with Terraform



\## Project



This project demonstrates how to build secure Azure infrastructure on Microsoft Azure using Terraform.



The objective is to learn Infrastructure as Code (IaC) by designing, deploying, and managing cloud infrastructure through code instead of manual Azure Portal configuration.



\## Scenario



A startup wants to deploy a new internal application on Azure.



Requirements:



\* Infrastructure must be reproducible.

\* Networking must be secure.

\* Everything should be version controlled.

\* No manual Azure Portal configuration.

\* Deployment should be automated.
